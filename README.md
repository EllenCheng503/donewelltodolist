# DoneWell To-Do List

DoneWell To-Do List is a browser-based personal task manager designed to help users organize daily work with simple structure and visual priority planning.

## What it does

This project helps users:

- Add tasks with descriptions and due dates
- Organize work by priority using a 4-quadrant priority matrix
- Set a personal role context during setup
- Mark different task types
- Back up and restore tasks
- Automatically save tasks in the browser

## Why it is useful

Many lightweight to-do tools are either too simple or too overloaded. DoneWell focuses on a clean, single-page experience for personal productivity and quick prioritization without requiring account registration or server setup.

## Features

- Personal welcome/setup screen
- Task description + due date
- Priority levels:
  - Urgent & Important
  - Urgent, Not Important
  - Not Urgent, Important
  - Not Urgent, Not Important
- Team task / partner task classification
- Today & Upcoming view
- Priority Matrix view
- Local browser autosave
- Backup / Restore support

## Tech stack

- HTML
- Client-side browser storage
- Single-file frontend structure

## How to use

1. Open `donewell.html` in your browser
2. Enter your name and role
3. Add tasks with due dates and priority
4. Review tasks in Today & Upcoming and Priority Matrix
5. Use backup and restore when needed
